# Justin Bieber — Fan Experience

An immersive, animation-driven single-page tribute to Justin Bieber, built as a
front-end **portfolio centerpiece**. Cinematic, editorial, and kinetic — designed
to feel like an "Awwwards Site of the Day".

> Fan-made portfolio project. Not affiliated with the artist.

## Tech stack

- **React 18 + TypeScript + Vite**
- **Tailwind CSS v4**
- **Motion** (Framer Motion) — reveals, parallax, gestures, layout animations
- **Lenis** — momentum smooth scrolling
- **lucide-react** — icons

## Highlights

- Animated **preloader** with a counter + name reveal
- **Custom cursor** and **magnetic buttons**
- **Lenis smooth scroll** (auto-disabled for `prefers-reduced-motion`)
- Top **scroll-progress bar** + sticky nav with **active-section** highlight
- Working fullscreen **mobile menu** (circular reveal)
- **Hero** with animated aurora glow + parallax
- **About** with scroll-triggered **count-up** stats
- **Discography** as a scroll-driven **horizontal pin** gallery
- Animated **timeline**, dual-direction **marquee** gallery, and tour list
- Fully **responsive**, accessible focus states, reduced-motion support

## Running the code

```bash
npm install     # install dependencies
npm run dev     # start the dev server (http://localhost:5173)
npm run build   # production build into /dist
```
